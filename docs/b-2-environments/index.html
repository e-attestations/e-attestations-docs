Environments

e-Attestations EDGE API provides two environments

EnvironmentURLDescription
TESThttps://edge.test-e-attestations.com/api/v1You'll use this env to develop, in a fully secure manner, on identical infrastructures than in production. Note that data is anonymized and no emails will be emitted.
PRODUCTIONhttps://edge.e-attestations.com/api/v1The production environnement.
caution

In the TEST environment emails are not sent (no invitation, no reports etc.) to thirdparties.

Keep this in mind, and feel free to contact us if you have specific needs.
