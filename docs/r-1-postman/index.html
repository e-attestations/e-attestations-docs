To go even faster and facilitate the exploration and the implementation of our EDGE API and exploration of our API we provide you with a pretty huge Postman collection that you can find here

You can explore many scenarios and use cases to functionally understand the infinite possibilities.

Using Postman 'environnement'#

When you use Postman collection, we advise you to use environnement which is a set of variables.

To use this test suite, you'll have to set :

  • username
  • password
  • ea_auth_serv
  • ea_api_serv
