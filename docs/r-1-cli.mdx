The e-Attestations Command Line Interface is a tool that you can use as a developper on any terminal.

It offers commands per resources and, Of course, interacts with 'EDGE' API.

This is a open Source project, written in Python and published in GitHub [here](https://github.com/e-attestations/e-attestations-cli)
