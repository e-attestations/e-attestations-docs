e-Attestations EDGE API provides two environments

Environment | URL | Description
------------|-----|---------------------------------
TEST|https://edge.test-e-attestations.com/api/v1|You'll use this env to develop, in a fully secure manner, on identical infrastructures than in production. Note that data is anonymized and no emails will be emitted.
PRODUCTION|https://edge.e-attestations.com/api/v1|The production environnement.

:::caution
In the TEST environment emails are not sent (no invitation, no reports etc.) to thirdparties.

Keep this in mind, and feel free to contact us if you have specific needs.
:::
